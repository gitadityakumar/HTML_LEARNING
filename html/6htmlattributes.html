Html attributes

Global attributes that can be applied to any HTML element common attributes

The class Attribute

Allows us to target all elements with that class in our CSS or JavaScript

<p class=" intro ">
this is the introduction.
</p>

The id Attribute

Allows us to target a unique element with that id in our CSS and JavaScript
<p class=" intro " id="article-intro">
this is the introduction.
</p>
